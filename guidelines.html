MusicBrainz ID

With each submission to the database, you must include the release's MusicBrainz ID. Make sure you select the correct issue for your upload!

Sources

Mention the website, tracker or method through which you obtained the audio files. We expect you to be honest, as this helps users discern sources with lossy transcodes. It's okay to leave this empty if you don't know the source.

Uploads

All audio and image file types are allowed, plus .txt files for additional information regarding a release. Providing cue/log files for CD rips is appreciated, as it helps verify the quality of your upload. For vinyl rips, include the lineage (equipment used to rip the record). Your audio files will not be stored after being spectographed and verified (okay, FBI?).
