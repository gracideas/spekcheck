What is a MusicBrainz ID?

Source refers to how and where you obtained the audio files. We expect you to be honest!

File types accepted: all audio formats, all image formats, .txt files and cue/log. Providing cue/log (CD) or lineage (Vinyl) is greatly appreciated.
